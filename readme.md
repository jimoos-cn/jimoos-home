## 构建说明

使用 基于`tailwind css` 构建的。

> tailwind css地址: https://tailwindcss.com 

修改`index.html`后，打包 `css`.

执行 `css` 打包命令。

`NODE_ENV=production npx tailwindcss-cli@latest build ./src/css/tailwind.css -o ./src/css/t-tailwind.css`
